Задание

  1. Изучите информацию о mc, вызвав в командной строке man mc.
  2. Запустите из командной строки mc, изучите его структуру и меню.
  3. Выполните несколько операций в mc, используя управляющие клавиши (операции с панелями; выделение/отмена выделения файлов, копирование/перемещение фай- лов, получение информации о размере и правах доступа на файлы и/или каталоги и т.п.)
  4. Выполните основные команды меню левой (или правой) панели. Оцените степень подробности вывода информации о файлах.
  5. Используя возможности подменю Файл , выполните: – просмотр содержимого текстового файла; – редактирование содержимого текстового файла (без сохранения результатов редактирования); – создание каталога; – копирование в файлов в созданный каталог.
  6. С помощью соответствующих средств подменю Команда осуществите: – поиск в файловой системе файла с заданными условиями (например, файла с расширением .c или .cpp, содержащего строку main); – выбор и повторение одной из предыдущих команд; – переход в домашний каталог; – анализ файла меню и файла расширений.
  7. Вызовите подменю Настройки . Освойте операции, определяющие структуру экрана mc (Full screen, Double Width, Show Hidden Files и т.д.)ю 7.3.2. Задание по встроенному редактору mc
  8. Создайте текстовой файл text.txt.
  9. Откройте этот файл с помощью встроенного в mc редактора.
  10. Вставьте в открытый файл небольшой фрагмент текста, скопированный из любого другого файла или Интернета.
  11. Проделайте с текстом следующие манипуляции, используя горячие клавиши: 4.1. Удалите строку текста. 4.2. Выделите фрагмент текста и скопируйте его на новую строку. 4.3. Выделите фрагмент текста и перенесите его на новую строку. 4.4. Сохраните файл. 4.5. Отмените последнее действие. 4.6. Перейдите в конец файла (нажав комбинацию клавиш) и напишите некоторый текст. 4.7. Перейдите в начало файла (нажав комбинацию клавиш) и напишите некоторый текст. 4.8. Сохраните и закройте файл.
  12. Откройте файл с исходным текстом на некотором языке программирования (напри- мер C или Java)
  13. Используя меню редактора, включите подсветку синтаксиса, если она не включена, или выключите, если она включена

Ответы на вопросы

Вопрос 1. Какие режимы работы есть в mc? Охарактеризуйте их.

Ответ:

В Midnight Commander существуют следующие основные режимы отображения панелей:

Режим Характеристика
Стандартный Выводит список файлов и каталогов с указанием размера и времени последней модификации
Ускоренный Позволяет задать число столбцов, на которые разбивается панель; выводит только имена файлов без дополнительной информации
Расширенный Помимо имени файла выводит права доступа, владельца, группу, размер, время правки
Информация Выводит сведения о текущем файле (размер, права, владелец) и о файловой системе (свободное место и т.д.)
Дерево Отображает структуру дерева каталогов файловой системы

Переключение режимов осуществляется через меню F9 → Левая панель / Правая панель.

Вопрос 2. Какие операции с файлами можно выполнить как с помощью команд shell, так и с помощью меню (комбинаций клавиш) mc? Приведите несколько примеров.

Ответ:

Операция Команда shell Команда / клавиши mc
Копирование файла cp source dest F5
Перемещение файла mv source dest F6
Удаление файла rm file F8
Создание каталога mkdir dirname F7
Просмотр файла cat file, less file F3
Редактирование файла nano file, vim file F4
Переименование mv old new F6 (ввести новое имя)
Изменение прав доступа chmod mode file Ctrl+x → c (Права доступа)

Вопрос 3. Опишите структуру меню левой (или правой) панели mc, дайте характеристику командам.

Ответ:

Меню Левая панель (аналогично — Правая панель) вызывается через F9 → Левая панель. Содержит следующие подпункты:

Команда Характеристика
Список файлов Стандартный режим отображения списка файлов
Быстрый просмотр Позволяет быстро просмотреть содержимое выбранного файла без открытия полноценного просмотрщика
Информация Переключает панель в режим отображения информации о файле и файловой системе
Дерево Переключает панель в режим отображения дерева каталогов
Формат списка Позволяет выбрать формат: стандартный, ускоренный, расширенный, пользовательский
Сортировка Задаёт порядок сортировки файлов (по имени, расширению, времени, размеру и т.д.)
Фильтр Позволяет отображать только файлы, соответствующие заданной маске (например, *.txt)
Переставить панели Меняет местами левую и правую панели

Вопрос 4. Опишите структуру меню Файл mc, дайте характеристику командам.

Ответ:

Меню Файл вызывается через F9 → Файл. Содержит команды для работы с выделенными файлами и каталогами:

Команда (клавиша) Характеристика
Просмотр (F3) Просмотр содержимого файла без возможности редактирования
Правка (F4) Редактирование файла во встроенном редакторе
Копирование (F5) Копирование выделенных файлов/каталогов в каталог, отображаемый на второй панели
Перемещение (F6) Перемещение (вырезать + вставить) выделенных файлов/каталогов
Создание каталога (F7) Создание нового подкаталога в текущем каталоге
Удалить (F8) Удаление выделенных файлов/каталогов
Символическая ссылка Создание символической ссылки на текущий/выделенный файл
Жёсткая ссылка Создание жёсткой ссылки на текущий/выделенный файл
Права доступа Изменение прав доступа (chmod) для файлов/каталогов
Владелец/группа Изменение владельца и группы для файлов/каталогов
Переименование Переименование или перемещение файла/каталога
Выход (F10) Завершение работы mc

Вопрос 5. Опишите структуру меню Команда mc, дайте характеристику командам.

Ответ:

Меню Команда вызывается через F9 → Команда. Содержит общие команды для работы с mc и файловой системой:

Команда (клавиша) Характеристика
Дерево каталогов Отображает структуру каталогов системы в отдельном окне
Поиск файла Поиск файлов по имени, содержимому, размеру, времени модификации
Переставить панели Меняет местами левую и правую панели (аналог Ctrl+u)
Сравнить каталоги Сравнивает содержимое двух каталогов, отображаемых на панелях
Размеры каталогов Вычисляет и отображает реальный размер каталогов (по умолчанию mc размер не показывает)
История командной строки Показывает список ранее введённых команд оболочки
Каталоги быстрого доступа Позволяет быстро перейти в один из заданных каталогов (Ctrl+\)
Восстановление файлов Восстановление удалённых файлов на файловых системах ext2/ext3
Редактировать файл расширений Настройка действий для файлов с определёнными расширениями
Редактировать файл меню Редактирование пользовательского меню (вызываемого по F2)
Редактировать файл расцветки имён Настройка цветового выделения файлов в зависимости от типа

Вопрос 6. Опишите структуру меню Настройки mc, дайте характеристику командам.

Ответ:

Меню Настройки вызывается через F9 → Настройки. Позволяет настраивать внешний вид и поведение mc:

Команда Характеристика
Конфигурация Основные настройки: подсветка синтаксиса, автопросмотр файлов, поведение клавиш и т.д.
Внешний вид Настройка отображения элементов интерфейса (меню, командная строка, подсказки, цветовые схемы)
Настройки панелей Геометрия панелей, тип отображения имён (нижний/верхний регистр), показ скрытых файлов
Биты символов Настройка кодировки и формата обработки информации локальным терминалом
Подтверждение Включение/отключение запросов подтверждения при удалении, перезаписи и выходе
Распознание клавиш Тестирование и настройка функциональных клавиш и клавиш управления курсором
Виртуальные ФС Настройка виртуальной файловой системы (тайм-аут, пароль для удалённых подключений)

Вопрос 7. Назовите и дайте характеристику встроенным командам mc.

Ответ:

Встроенные команды mc — это команды, которые можно вводить в командной строке mc (строка внизу экрана, начинается с «cd » или просто команда). Основные встроенные команды:

Команда Характеристика
cd <путь> Сменить текущий каталог на активной панели
cd .. Перейти на уровень вверх
cd / Перейти в корневой каталог
cd ~ Перейти в домашний каталог
pwd Показать текущий рабочий каталог
ls Вывести список файлов (вывод появляется после отключения панелей Ctrl+o)
find Поиск файлов (работает как обычная команда shell)
grep Поиск строк в файлах
!<команда> Выполнить команду shell без выхода из mc (например, !ls -la)

Также в командной строке работает история (стрелка вверх/вниз) и автодополнение по Tab.

Вопрос 8. Назовите и дайте характеристику командам встроенного редактора mc.

Ответ:

Встроенный редактор mc вызывается по F4. Основные команды редактирования:

Команда (клавиша) Характеристика
Ctrl+y Удалить текущую строку
Ctrl+u Отменить последнее действие
Ins Переключение режимов вставка/замена
F7 Поиск текста (можно использовать регулярные выражения)
Alt+F7 Повтор последнего поиска
F4 Замена текста
F3 Начало выделения (первое нажатие), конец выделения (второе нажатие)
F5 Копировать выделенный фрагмент
F6 Переместить выделенный фрагмент
F8 Удалить выделенный фрагмент
F2 Сохранить файл
F10 Выйти из редактора
Ctrl+Home Перейти в начало файла
Ctrl+End Перейти в конец файла

Вопрос 9. Дайте характеристику средствам mc, которые позволяют создавать меню, определяемые пользователем.

Ответ:

Пользовательское меню в mc вызывается по клавише F2. Его можно редактировать через:

F9 → Команда → Редактировать файл меню

Файл меню обычно находится в ~/.config/mc/mc.menu (или ~/.mc/menu в старых версиях).

Характеристика средств:

Возможность Описание
Синтаксис файла Меню описывается в текстовом файле с использованием специальных меток
Заголовок меню Задаётся строкой, начинающейся с буквы и пробела (например, A Проект)
Буква быстрого вызова Каждому пункту можно назначить букву (регистр имеет значение)
Команды После заголовка пишется команда shell, которая выполнится при выборе пункта
Условные пункты Пункт может отображаться только при определённых условиях (расширение файла, тип, права)
Подменю Возможно создание вложенных меню
Пример A Архивация\n tar -czf backup.tar.gz %f — при нажатии F2 → A архивирует текущий файл

Это позволяет пользователю добавить собственные часто используемые операции (архивация, компиляция, конвертация и т.д.).

Вопрос 10. Дайте характеристику средствам mc, которые позволяют выполнять действия, определяемые пользователем, над текущим файлом.

Ответ:

Основным средством для выполнения пользовательских действий над текущим файлом является файл расширений.

Файл расширений редактируется через: F9 → Команда → Редактировать файл расширений (обычно ~/.config/mc/mc.ext)

Характеристика средств:

Возможность Описание
Привязка к расширению Определяет, какое действие выполнить при нажатии Enter на файле с определённым расширением (например, .txt → открыть в редакторе, .jpg → показать картинку)
Привязка к типу файла Можно задать действия для каталогов, исполняемых файлов, ссылок и т.д.
Секции Файл разбит на секции: [.txt], [.c], [.jpg], [directory], [regexp] и т.д.
Макросы Используются специальные переменные: %f (текущий файл), %d (текущий каталог), %p (выделенные файлы)
Команды для просмотра Задаются директивой View= (вызывается по F3)
Команды для редактирования Задаются директивой Edit= (вызывается по F4)
Команды по умолчанию Задаются директивой Open= (вызывается по Enter)
Пример [.c]\n Open=gcc %f -o %f.out\n View=cat %f — для .c файлов: Enter скомпилирует, F3 покажет содержимое

Выводы

  1. Midnight Commander — мощная псевдографическая командная оболочка, упрощающая работу с файловой системой в UNIX/Linux.

  2. Управление mc осуществляется с помощью функциональных клавиш F1–F10, комбинаций клавиш и меню, вызываемого по F9.

  3. Панели могут работать в пяти режимах: стандартный, ускоренный, расширенный, информация и дерево.

  4. Основные операции с файлами (копирование, перемещение, удаление, редактирование) выполняются быстрее, чем через shell-команды.

  5. Встроенный редактор mc позволяет редактировать текстовые файлы с поддержкой поиска, замены, выделения, копирования и отмены действий.

  6. Средства mc дают возможность настраивать поведение программы: изменять внешний вид, создавать пользовательские меню и задавать действия для файлов по расширениям.

  7. Изученные навыки работы с mc ускоряют выполнение повседневных задач администратора и пользователя Linux.
